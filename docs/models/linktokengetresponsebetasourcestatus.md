# LinkTokenGetResponseBetaSourceStatus

The beta source status filter from which data sources will be displayed during the Link session corresponding to the provided Link token.


## Values

| Name            | Value           |
| --------------- | --------------- |
| `LIVE_AND_BETA` | LiveAndBeta     |
| `ONLY_LIVE`     | OnlyLive        |
| `ONLY_BETA`     | OnlyBeta        |
